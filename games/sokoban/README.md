# 倉庫番（Sokoban）

ランダム生成の倉庫番パズルです。**逆生成**（解けた配置から箱を引いて混ぜる）により、必ずクリア可能なステージだけを生成します。  
生成後は **ゴール外に箱が 1 個以上ある** ことも検証し、「運ぶ対象が無い」ステージは除外します。

## SD 配置

```
/games/sokoban/sokoban.lua
```

任意:

- `fonts/game_font.bin` … タイトルに「倉庫番」を表示（他ゲームと同じ MISF フォント）
- `sokoban.bin` または `title.bin` … ゲーム選択メニューのプレビュー（100×100 RGB565）

## 操作

| ボタン | 動作 |
|--------|------|
| LEFT / RIGHT（タイトル） | 難易度選択 |
| NEAR 等（決定） | タイトルで開始 / クリア画面をスキップ |
| 十字キー（プレイ中） | 移動・箱を押す |
| OP_LEFT | プレイ中にタイトルへ |

## 難易度

| 難易度 | 柱（障害物）の数 | 初期箱数 | 得点倍率 |
|--------|------------------|----------|----------|
| EASY | 2〜3 | 2 から | ×1.0 |
| NORMAL | 4〜6 | 2 から | ×1.25 |
| HARD | 7〜10 | 3 から | ×1.5 |

ステージが進むほど箱の数と混ぜ回数が増えます（HARD は最大 6 箱）。

## 得点・ハイスコア

ステージクリアごとに加算（難易度倍率あり）:

- 基本点: `100 × ステージ番号`
- 手数ボーナス: `max(0, 400 − 手数 × 4)`
- 箱ボーナス: `箱の数 × 50`

**難易度ごとに** ハイスコアを `hi_score.dat` に保存します（`scores = { easy, normal, hard }`）。  
旧形式（単一 `hi_score`）は NORMAL へ移行されます。

## ステージ生成

1. 壁で囲んだ部屋＋難易度に応じたランダムな柱
2. ゴール位置を箱の数だけ配置
3. 全箱をゴール上に置いた「完成状態」から開始
4. 逆操作（箱を引く／歩く）を繰り返して混ぜる
5. **検証**: 箱数が正しいこと、ゴール外の箱が 1 個以上あること
