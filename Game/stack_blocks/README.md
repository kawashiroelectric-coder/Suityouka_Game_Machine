# Stack Blocks（落ち物パズル）

シンプルな落ち物パズルのサンプルです。商標上の固有名詞は使用していません。

## SD 配置

```
/games/stack_blocks/
└── stack_blocks.lua
```

フォルダ名と同名の `stack_blocks.lua` が起動スクリプトとして自動選択されます。

## 操作

| 入力 | 動作 |
|------|------|
| LEFT / RIGHT | 左右移動 |
| DOWN | ソフトドロップ |
| OP_LEFT | ハードドロップ（一気に落下） |
| UP / NEAR / OP_RIGHT | 回転 |
| ゲームオーバー後（ジャンプ系ボタン） | リトライ |

## ルール

- 横一列が埋まると消えてスコア加算
- 10 ライン消去ごとにレベルアップ（落下速度アップ）
- 新しいピースを置けなくなるとゲームオーバー
